How to play

If a number is dividable by the given ruleset then add the word to the prompt.

The buttons for the words are toggleable so if you make a mistake you can correct it.

The words don't need to be in any order.

FizzBuzz

If a number is dividable by 3 then it's Fizz.
If a number is dividable by 5 then it's Buzz.
If they are dividable by both then it's FizzBuzz.

FooBar

If a number is dividable by 7 then it's Foo.
If a number is dividable by 11 then it's Bar.
If they are dividable by both then it's FooBar.

FizzBuzzFooBar

Just the same rules as FizzBuzz and FooBar but this time combined!
